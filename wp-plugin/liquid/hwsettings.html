WordPress settings


Base URL:
Your base URL should look like this https://your-username.wordpress.com. If you have your own WordPress server then your base URL may be http://www.yourserver.com/ or http://www.yourserver.com/wordpress depending on the configuration of your server.



WordPress post URL suffix
Different WordPress versions use for blog posting either 'post.php' file 'post-new.php' file. You should select the correct file name to enable WordPress support in the Hyperwords.
